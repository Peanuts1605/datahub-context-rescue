## Messy Data-Team Question

The weekly revenue dashboard dropped after yesterday's customer/orders change.
What changed, who owns it, what is affected, and what should we do next?

## Expected Agent Job

Use DataHub context before answering:

1. Search for the revenue/dashboard cluster.
2. Inspect the matching entities.
3. Pull schema fields and metric definitions.
4. Pull upstream and downstream lineage.
5. Identify owners or escalation contacts.
6. Return one Context Rescue Card with a NEXT/HOLD decision and a receipt.
